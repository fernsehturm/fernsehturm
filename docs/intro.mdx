# Introduction

Fernsehturm (FST) is a control system for AI-assisted software development.

You keep the product direction. Your agent does the work. FST keeps the work bounded, traceable, and checked before the agent moves forward.

> Build Software with AI without Losing Control.

## How It Works

FST controls four stages, each with a strict exit gate:

```text
Work-Context Selection → Exploration → Build → Compose
```

The agent is free inside each stage — inspecting, experimenting, testing, revising. FST checks whether the result is ready to leave the stage.

## What You Do

You do five things:

1. Say what you want.
2. Confirm the starting context when there is more than one sensible option.
3. Answer decisions that affect behaviour, scope, policy, or risk.
4. Approve scope changes or materialization choices.
5. Review the trace-backed result.

Everything else — implementation, verification, coherence checking — the agent and FST handle.
