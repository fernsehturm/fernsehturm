# Installation

## Install Fernsehturm

```bash
npm install -g fernsehturm
```

Verify the installation:

```bash
fst --version
```

---

## Initialize a project

Run this once in the root of your project:

```bash
fst self init
```

This creates the Fernsehturm database and workspace structure. After this, `fst` is the entry point for all work in this project.

---

## What gets created

| Path | Purpose |
|------|---------|
| `.fst/db.sqlite` | the database — source of truth for all artifacts |
| `.fst/workspace/` | generated files — do not edit directly |
| `.fst/config.json` | project configuration |

The workspace is a derived projection of the database. It will be overwritten. Do not treat it as a place to make changes.
